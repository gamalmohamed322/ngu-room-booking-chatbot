# NGU Room Booking Chatbot

An AI-powered room booking assistant designed for New Giza University students.

The chatbot helps students book study rooms, meeting rooms, or lab rooms by collecting booking details such as room type, date, time, student name, email, and phone number.

## Project Idea

Many students need to reserve rooms for studying, meetings, club activities, or group projects. This chatbot makes the booking process easier by allowing students to request a room through a conversational interface.

## Features

- Chatbot-based room booking flow
- Student information collection
- Room type selection
- Date and time selection
- Booking confirmation
- Backend webhook integration
- Email notification system
- Booking conflict validation
- SQLite database storage

## Tech Stack
